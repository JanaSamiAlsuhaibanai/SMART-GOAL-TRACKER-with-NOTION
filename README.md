# Smart Goal Tracker with Notion

Smart Goal Tracker with Notion is a web application that helps users plan and track daily goals. It uses AI to suggest optimal times for activities based on your routine and energy levels, then automatically adds tasks to your Notion database. The app features a modern interface, daily schedule visualization, and seamless Notion integration for efficient goal management.

## Features
- Add and manage daily goals
- AI-powered time suggestions based on your routine
- Automatic task creation in Notion
- Visualize your daily schedule and available time slots
- Customize your sleep, work, commute, and energy levels

## Getting Started
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the app:
   ```bash
   streamlit run app.py
   ```

## Configuration
- Add your Notion API token and database ID in the code or via environment variables.
- Add your Cohere API key for AI-powered suggestions.
